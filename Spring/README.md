## TERASOLUA

[본문](http://terasolunaorg.github.io/)

TERASOLUNA는 NTT데이터가 제공하는 「프레임워크」, 「개발 프로세스」, 「프로젝트 관리」 등의 기술이나 노하우를 조합하여, 시스템 개발을 포괄적으로 서포트하는 솔루션입니다. NTT 데이터는 지금까지 TERASOLUNA를 이용하여 대규모적이며 복잡한 미션 크리티컬한 시스템을 많이 구축해왔습니다. NTT 데이터에서는 TERASOLUNA를 이용, 성장시키는 것으로, 시스탬 개발의 모습을 발본적(좋지 않은 일의 근본 원인이 되는 요소를 완전히 없애 버림)으로 혁신시킵니다. 

## TERASOLUNA Server Framework for Java (5.x)

높은 신뢰성이 요구되는 Enterprise 개발에 있어 세계적으로 넓게 사용되는 Spring Framework 등의 오픈소스, 소프트웨어(이하 OSS)를 최대한 활용하여, NTT 데이터가 오랜 경험으로 만든 노하우를 활용한 플랫폼입니다. 이런 OSS의 활용에 있어, 최신 기술의 추종이나 글로벌한 개발자 확보가 쉬워지고, 당사만이 아니라 전 세계의 개발자의 의견이나 노하우를 활용하는 것이 가능합니다.
