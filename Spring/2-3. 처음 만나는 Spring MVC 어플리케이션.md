## GET, POST

- 데이터를 서버에 보존하는 경우(갱신 등의 처리의 경우)는 POST가 좋다
- 데이터를 서버에 보존하지 않는 경우(참조 등의 처리의 경우)는 GET 또는 미지정(임의의 메소드)가 좋다.

## @ModelAttribute

- @ModelAttribute 태그를 붙인 메소드의 반환값은 자동으로 Model에 저장된다.

    		@ModelAttribute // (1)
        public EchoForm setUpEchoForm() {
            EchoForm form = new EchoForm();
            return form;
        }
    
        @RequestMapping // (2)
        public String index(Model model) {
            return "echo/index"; // (3)
        }

setUpEchoForm의 반환값 form이 자동으로 index 메소드의 Model에 저장된다.

## @NotNull

HTTP 리퀘스트 중 파라미터가 있는지를 확인한다

## @Size

@Size(min =1, max =5)를 붙이면, 파라미터의 사이즈가 1이상 5이하인지를 확인한다.
